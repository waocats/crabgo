# crabgo-help(1)

## NAME

crabgo-help --- Get help for a Crabgo command

## SYNOPSIS

`crabgo help` [_subcommand_]

## DESCRIPTION

Prints a help message for the given command.

## EXAMPLES

1. Get help for a command:

       crabgo help build

2. Help is also available with the `--help` flag:

       crabgo build --help
